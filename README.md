# Australian Census Ancestry Analysis (2016–2021)

## Overview
This project analyses demographic shifts in Australia using ABS Census DataPacks (2016 and 2021). The focus is on ancestry trends, growth rates, and structural population change.

## Key Findings
- English and Australian ancestries remain the largest reported groups.
- Indian and Chinese ancestries recorded significant proportional growth.
- Continued diversification of Australia’s demographic composition is evident between 2016 and 2021.

## Methodology
- Cleaning and harmonisation of ABS G08 DataPack tables
- Cross-year alignment of ancestry categories
- Growth rate calculation
- Comparative visual analysis of top 10 reported ancestries

## Tools Used
Python (pandas, numpy, matplotlib)

- `Australian_Census_Ancestry_Analysis.ipynb` — Full reproducible analysis
- `Australian_Census_Ancestry_Analysis.html` — Rendered analysis report
